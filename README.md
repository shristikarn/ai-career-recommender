# AI Career Recommender System

## Overview
An *AI-based career recommendation system* that suggests the best career paths, detects skill gaps, generates personalized learning roadmaps, and recommends projects based on user skills and interests. Built with *Python* and *MySQL*.


## Features
- Career recommendation based on *user skills, interests, and experience level*
- Skill gap analysis to highlight missing skills
- Step-by-step roadmap generator for learning and career growth
- Project suggestions to apply skills in real-world scenarios
- Personalized roadmap prioritization based on user profile


## Tech Stack
- *Backend:Python
- *Database:MySQL
- *Libraries:mysql-connector-python
- *Data:100+ entries in careers, skills, roadmap, and projects
